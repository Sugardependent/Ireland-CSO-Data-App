CSO Data App

This demo application uses ChartJS to display census Data from the Central Statistics Office regarding the Irish Population.

Stack:
  • Angular 2 Frontend with SASS, Bootstrap and ChartJS
  • Django Rest Framework Backend using Postgresql Database

The data displayed has been sourced from the Central Statistics Office. As CSO data may be subject to change or corrections over time, consider the data shown here as example data sampled in August 2017 which may be subject to errors or inconsistencies.
